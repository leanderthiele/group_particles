Detailed Description

template<typename ReturnT, typename... ArgT>
class CallbackUtils::MultiAction::AbstractBase< ReturnT, ArgT >

base class that collects the actions to be performed.

Template Parameters
ReturnTfunction return value type
ArgTfunction argument types

The functions put into the container should have the signature ReturnT (void *, ArgT...), where the first argument can be used to pass additional information (usually an object).

Use this class by inheriting from it and Callback simultaneously and overriding the desired method in Callback using the provided call method. This produces a ConcreteBase-type, which can then be passed as template parameter to the #ConcreteInterface.

Attention
do not rely on the order of execution of the actions collected in the container.
