Detailed Description

template<typename AFields, typename VField, bool sqrta>
class CallbackUtils::prt_modify::PrtRSD< AFields, VField, sqrta >

Implement RSD shifting of particle positions along a coordinate.

Template Parameters
VFieldthe type of the velocity field (must be present).
sqrtafor Gadget-descendent simulations, the velocities must be multiplied by sqrt(a) to get physical values.
