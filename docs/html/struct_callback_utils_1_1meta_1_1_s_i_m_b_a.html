Detailed Description

template<typename AFields, uint8_t PartType>
struct CallbackUtils::meta::SIMBA< AFields, PartType >

retrieves meta-data from a SIMBA-type simulation.

This is essentially the same as for an Illustris type sim (i.e. CallbackUtils::meta::Illustris), but with the subtle difference that NumPart_ThisFile is stored as a uint32_t
