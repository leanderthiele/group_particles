Detailed Description

template<typename AFields, typename Field, typename storeasT = typename Field::value_type>
class CallbackUtils::grp_action::StoreGrpProperty< AFields, Field, storeasT >

implements the common case of storing a single group property for each group.

Template Parameters
Fieldthe group property that is stored for each group
storeasTthe type the property should be cast to
Note
Currently, only 1-dimensional Fields are supported (and storeasT must be arithmetic)
